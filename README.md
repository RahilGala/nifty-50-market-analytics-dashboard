## How to Use

1. **View Overall Performance**: Check the top summary cards for quick market insights
2. **Analyze Trends**: Use the profit-wise chart to understand historical performance
3. **Sector Analysis**: Review sector-wise and industry-wise distributions to identify leading sectors
4. **Stock Selection**: Use the detailed stock table with filters to find specific stocks
5. **Risk Assessment**: Utilize volatility and ROE classifications for risk evaluation
6. **Reset Filters**: Click the "RESET" button in the left panel to clear all applied filters
